# Synchronous_FIFO
This project implements a Synchronous FIFO (First-In First-Out) buffer using Verilog HDL.
A synchronous FIFO uses a single clock for both read and write operations. It is commonly used in digital systems to temporarily store data between modules operating in the same clock domain.

The design includes:
FIFO memory
Read and write pointers
Full and empty status flags
Testbench for simulation
